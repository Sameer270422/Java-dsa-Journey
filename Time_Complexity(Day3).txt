1. No loop: O(1).:  int x = arr[0];
2. One loop: O(N).   
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
3. LOOP inside loop O(N^2).
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        System.out.println(i + j);
    }
}
4. Half TRaversal o(N)
for (int i = 0; i < n; i += 2)(O(N)We ignore constants and lower-order terms in Big-O.)

5. Drop constants O(N)


When two loops run sequentially from front and Back TC will be O(N)
